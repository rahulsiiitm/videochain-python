# VidChain: Technical Specifications (v0.9.1-Stable)

## 1. Engine Core (IRIS 2.0)
- **Architecture**: Evolved Asynchronous sensory matrix with Neural Isolation.
- **Intelligence Agent**: IRIS (Intelligent Retrieval & Insight System).
- **VLM Pipeline**: Moondream (Optimized Edge) or LLaVA 1.6 via Ollama.
- **RAG Subsystem**: ChromaDB (Vector) + NetworkX Temporal Knowledge Graph.
- **Isolation Protocol**: Per-video isolated context files (.pkl) for high-integrity reasoning.
- **The Neural Lens**: Automated forensic snapshot extraction via OpenCV with Base64 encoding for visual proof.
- **Optimization**: Adaptive Gaussian differential keyframing with 7s neural warmup and 900s "Infinite Patience" neural timeouts.

## 2. VidChain Intelligence Portal
- **Framework**: Next.js 14 (App Router) + Turbopack.
- **Styling**: Vanilla CSS + Framer Motion (Premium Glassmorphic Aesthetic).
- **Media Engine**: VidChain Media Gateway (FastAPI FileResponse). Supports absolute local paths for universal Windows playback.
- **Session Logic**: Deferred creation with robust state restoration and persistent chat history.

## 3. Hardware Requirements
- **Minimum**: 16GB RAM, 4GB VRAM (NVIDIA RTX 3050+).
- **Optimized**: 32GB RAM, 12GB VRAM (NVIDIA RTX 4070+).
- **Neural HUD**: Real-time chapter-level progress tracking and Hardware Telemetry (v0.9.1).
